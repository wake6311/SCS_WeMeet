# WE Meet STT Viewer

상담 인터뷰 STT 데이터를 바탕으로 학생의 진로 방향을 구조적으로 읽어내는 Streamlit 앱입니다.  
단순 뷰어가 아니라, **직무 추천 근거**, **역량 축**, **상담사 코멘트 품질**까지 같이 확인할 수 있도록 구성되어 있습니다.

## What It Does

- 인터뷰 JSON 목록 조회
- 전공/태그 기준 필터링
- 인터뷰 원문 대화 확인
- 직무 추천 구조 프레임워크 조회
- 인터뷰별 상위 추천 직무와 추천 근거 확인
- 역량 축 자동 추출
  - `분석형`
  - `설득형`
  - `조율형`
  - `실험형`
- 추천 사유 요약문 자동 생성
- 상담사 코멘트 품질 분석
  - 탐색
  - 공감
  - 구조화
  - 실행 제안
  - 균형
- 본문 검색 결과 확인

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

앱 실행 후 브라우저에서 Streamlit UI로 바로 확인할 수 있습니다.

## Main Tabs

### 1. 인터뷰 목록

현재 데이터셋의 인터뷰 메타데이터를 빠르게 훑어보는 탭입니다.

### 2. 대화 상세

인터뷰별 메타데이터와 전체 발화를 시간 순서대로 확인할 수 있습니다.

### 3. 추천 구조

직무 추천이 어떤 기준으로 계산되는지 프레임워크를 보여줍니다.

기준 축:

1. 전공 적합성
2. 명시적 관심사(`persona_tags`)
3. 본문 키워드 근거
4. 업무 방식 적합성
5. 역량 축 신호

### 4. 직무 추천 분석

학생별 상위 추천 직무, 추천 점수, 근거 발화, 역량 축, 자동 생성된 추천 요약문을 확인할 수 있습니다.

### 5. 상담사 코멘트 분석

상담사가 학생 이야기를 얼마나 잘 탐색하고, 공감하고, 구조화하고, 행동 제안까지 연결하는지를 정리합니다.

### 6. 본문 검색 결과

특정 표현이나 키워드가 포함된 발화를 현재 필터 범위 안에서 바로 찾을 수 있습니다.

## Data Format

각 인터뷰는 JSON 파일이며, 대략 아래 구조를 따릅니다.

```json
{
  "interview_id": "INT-001",
  "metadata": {
    "session_date": "2026-03-18",
    "duration": "00:20:51",
    "client": {
      "name": "김나은",
      "major": "영어영문학과",
      "persona_tags": ["브랜드 콘텐츠", "카피라이팅", "콘텐츠 에디터"]
    }
  },
  "utterances": [
    {
      "speaker": "C",
      "text": "안녕하세요. 편하게 앉으세요."
    },
    {
      "speaker": "A",
      "text": "졸업이 얼마 안 남았는데 방향이 잘 안 잡혀요."
    }
  ]
}
```

## Notes

- 추천 로직은 현재 규칙 기반입니다.
- `persona_tags`, 전공, 본문 키워드, 발화 패턴을 함께 사용합니다.
- 상담사 품질 분석도 규칙 기반 휴리스틱입니다. 절대적인 평가보다 비교/검토 용도로 보는 것이 적절합니다.
